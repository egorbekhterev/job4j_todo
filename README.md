# job4j_todo

## Описание проекта

* Web-приложение, представляющее собой "TODO список".

## Стек технологий

- **Java 17.0.2**
- **Spring Boot 2.7.6**
- **Thymeleaf**
- **Bootstrap 5.2.3**
- **HTML5**
- **PostgreSQL 15**
- **Hibernate 5.6.11**
- **Liquibase 4.15.0**
- **H2DB 2.1.214**
- **Commons-dbcp2 2.9.0**
- **Junit 5.8.2**
- **Mockito 3.5.13**
- **Maven 3.8.1**

## Требования к окружению

- **Java 17.0.2**
- **Maven 3.8.1**
- **PostgresSQL 15**

## Сборка и запуск

- **Создать БД**

``` 
create database todo;
```

- **Запустить проект по команде**

``` 
mvn spring-boot:run -Pproduction
```

- **Перейти в браузере по ссылке**

``` 
http://localhost:8080/
```
